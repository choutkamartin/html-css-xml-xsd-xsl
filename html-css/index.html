Popis BP

O čem bude

Moje bakalářská práce se bude zabývat vývojem platformy pro přípravu dat vhodných ke strojovému učení.

English

My bachelor thesis will deal with the development of a platform for preparation data suitable for machine learning.

O webové platformě

Tato webová platforma bude umožňovat to, že klienti budou zádavat do tohoto systému data, která budou chtít nějakým způsobem zpracovat. Může se jednat o označení nějakého objektu na obrázku, či označení toho, zda se text, který je na obrazovce jeví jako pozitivní či negativní.

English

This web platform will allow clients to type in this system the data they will want to process in some way. It can be a designation of an object in the image, or a designation whether the text on the screen appears positive or not negative.

Využité technologie

React.js

React je JavaScriptová knihovna pro tvorbu uživatelského rozhraní. Je vyvíjený Facebookem a komunitou samostatných vývojářů a společností. React může být využit jako základ pro tvorbu single-page nebo mobilních aplikací, protože je optimální pro práci s rychle se měnícími daty.

MongoDB

MongoDB je multiplatformní dokumentová databáze. Řadí se mezi NoSQL databáze a místo tradičních relačních databází využívajících tabulky používá dokumenty podobné formátu JSON a dynamické databázové schéma, které umožňuje vytváření a integraci dat pro aplikace jednodušeji a rychleji.

Node.js

Node.js je softwarový systém navržený pro psaní vysoce škálovatelných internetových aplikací, především webových serverů. Programy pro Node.js jsou psané v jazyce JavaScript, hojně využívající model událostí a asynchronní I/O operace pro minimalizaci režie procesoru a maximalizaci výkonu.

Zadání práce

Cíle

Hlavním cílem práce je vytvořit webovou platformu pro přípravu dat vhodných ke strojovému učení na základě označování dat.

Metodika

Na základě analýzy odborné literatury bude provedena syntéza získaných informací. Tyto informace poslouží ke zpracování teoretického východiska. Praktická část práce spočívá ve využití metodiky vývoje softwaru. Dle modelu vodopádu provedeme analýzu požadavků a navrhneme řešení. Implementace bude provedena v programovacím jazyce JavaScript a technologií s ním spojených. Po implementaci bude následovat verifikace, software bude otestován, zda vyhovuje požadavkům a pokud ano, software bude nasazen a udržován. Pomocí logické indukce vyvodíme obecný závěr ze získaných poznatků při vývoji softwaru, navrhneme, jakým směrem by se mohl ubírat a popíšeme, kdo by ho využíval.

Využití v praxi

Firmy

Webovou platformu by mohly využít všechny firmy, které se zaobírají umělou technologií. Takovýto projekt by našel spoustu příznivců v řadách autonomního řízení. Například takový Lyft, Uber, Tesla a podobně.
